让Google搜索到个人博客

发表于 2020-11-08 分类于 实用技巧

我曾天真地以为搜索引擎能够搜索到自己的博客，咳咳… 真是 too young too naive.

毕竟个人博客作为一个不知名小网站，若想被Google搜索到，需要手动将网址链接添加到Google搜索引擎的索引中。本篇文章就记录下如何让Google搜索到个人博客。

0.查看个人博客是否被收录

​ 那么恭喜你！你发现了一片宝藏文章，继续看下去吧。

else

​ 相逢即是缘…

1.打开Google Search Console

选择网址前缀，输入个人博客网址，比如我的https://lyh9909.github.io

点击继续，出现下图

  • 下载该html文件并上传至个人博客根目录https://lyh9909.github.io/
    • 以我个人博客本地目录为例，下载至：hexo/public
    • 提交至github仓库，可以看到https://lyh9909.github.io/ 里出现该html文件
  • 在浏览器中访问https://lyh9909.github.io/googled5a7a45251cda4f4.html 确认上传成功
  • 点击验证，出现下图

2.添加站点地图

网站地图描述了一个网站的架构，这有利于访问者以及搜索引擎找到网站的页面。

  • 打开 XML-sitemap , 输入博客地址，点击start

  • 等待搜索完成，点击 VIEW SITE DETAILS

  • 点击下载 sitemap.xml 文件并上传至个人博客根目录 (方法同前面的google.html文件)

3.提交站点地图

  • 再次打开Google Search Console

  • 点击主菜单中的站点地图

  • 在框里输入sitemap.xml，点击提交

  • 提交成功

嘿嘿，恭喜你，完结撒花~~

等等，回来！为何发现个人博客还是没有被收录？！

哎呦，不要着急嘛！要等google下一次更新索引的时候才会将你添加的内容收录进去滴 ~

大概等个2-3周吧，先把个人博客的内容优化一下吧，这么少的内容哪里有人看 (我说我自己…)
