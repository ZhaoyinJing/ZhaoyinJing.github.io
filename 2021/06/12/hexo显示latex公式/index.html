按照官网教程在hexo下实现latex公式渲染。

Hexo v4.2.0|NexT v7.7.1

首先强推官网教程，此前我曾在各种博客里搜寻解决办法，发现一点作用都没有。这告诫我们，遇到困难第一时间找官方手册解决问题。

具体步骤

  • 卸载hexo-math：$ npm un hexo-math

  • 替换渲染

    • $ npm un hexo-renderer-marked

    • $ npm i hexo-renderer-pandoc

      pandoc is required for hexo-renderer-pandoc, here's how to install pandoc.

  • 在hexo/_config.yml末尾添加一段代码

  • 修改next/_config.yml中mathjax的设置

注意：

如果per_page:false则md文件前面要加上mathjax:true

最后$ hexo clean && hexo g -d# or hexo clean && hexo s，检验是否成功。
